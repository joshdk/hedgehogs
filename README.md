## Installation

### Download the script

```
$ wget https://raw.githubusercontent.com/joshdk/hedgehogs/master/hedgehogs
```

### Copy into your `$PATH`

```
$ install hedgehogs ~/.bin/
$ which hedgehogs
  ~/.bin/hedgehogs
```

## Running

```
$ hedgehogs
```
